My projects

Over the years I've made countless projects, some ending up more finished than others. If you want a look into what I've done, you can see some of the highlights in the timeline below.
However, these are only a fraction of what I've done and I encourage you to check out my github if you want to see more.

My projects

  • 2018

    Console RPG

    One of my first major projects was a simple rpg for the cmd using python. Although I didn't finish it, it taught me alot about rendering.

  • 2019- 2021

    OpenGL fractals

    Rendering fractals using C++ OpenGL shaders has taught me alot about computer graphics and math- from probabilities to ray tracing and rasterization. My most ambitious project by far.

  • 2020

    Machine learning

    I took a dive into machine learning. It all begun with me making deepfakes using the first-order-model ai. After that, I tried making various projects, including one which would identify anime characters from an image with an average 98% accuracy.

  • 2021

    Cmd videos

    In a throwback to the past, I decided to make another project in the command prompt. Because of my improvements in programming, I was able to play videos seamlessly using only ascii characters.
